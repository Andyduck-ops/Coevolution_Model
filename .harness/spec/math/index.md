# Mathematical Specifications (RNA LLPS)

## P0 必须完整项（进入实现前）

1. `V_x, V_y` 明确进入参数表与复合参数计算。
2. 热力学参数组完整：`v_R, v_P, χ_RR, χ_PP, χ_0, ε, k_BT`。
3. 两相辅助函数链完整：`Θ_R, W^I/II, P_eff, A_eff, X_loc`。

## P1 强烈建议项

- `B_n = δ₀Eγ^n` 的下标约定与函数签名显式化。
- `Π_P = exp(ε x̄ȳ / k_BT)` 的数值稳定实现（对数域或裁剪策略）。

## 验收准则

- 方程条目需包含：符号定义、单位/无量纲、代码函数映射、测试映射。
- 禁止“只有标题没有公式/定义”的占位条目。
